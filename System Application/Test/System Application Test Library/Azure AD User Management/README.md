 for test purposes.

```
procedure Run(ForUserSecurityId: Guid)
```

*ForUserSecurityId ([Guid](https://docs.microsoft.com/en-us/dynamics365/business-central/dev-itpro/developer/methods-auto/guid/guid-data-type))*
